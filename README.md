# SCalendar
微信小程序校历组件  
导入微信开发者工具即可测试`DEMO`

```xml
<!-- 参数说明 -->
<calendar term="2019-2020-2" termStart="2020-02-10" weekCount="29" vacationStart="24"></calendar>
<!-- 
    term: 当前学期 
    termStart: 开学日期
    weekCount: 学期总周次
    vacationStart: 假期开始周次
-->
<!-- 注意: 所有填写日期需符合 yyyy-MM-dd 格式，例如 2020-03-01 而不是 2020-3-1 -->
```
